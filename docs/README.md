# Documentación de Referencia

Esta carpeta contiene la documentación base del Meta-Chamanismo que sirve como referencia para:

1. **Generación de contenido**: Material fuente para crear capítulos del seminario
2. **Asistencia con IA**: Documentación que utilizará el agente para recomendar mejoras al autor
3. **Consistencia conceptual**: Asegurar que el seminario mantenga coherencia con los principios del Meta-Chamanismo

## Archivos recomendados

Puedes colocar aquí:
- Documentos de Word (.docx)
- PDFs
- Archivos de texto (.txt, .md)
- Cualquier material de referencia sobre Meta-Chamanismo

## Uso

La documentación en esta carpeta será utilizada por agentes de IA para:
- Sugerir contenido relevante
- Mejorar la redacción manteniendo coherencia conceptual
- Generar capítulos basados en el material de referencia
- Validar que el seminario se alinee con los principios documentados
